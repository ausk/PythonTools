## PythonTools
使用 Python 制作的一些辅助小工具，不定期更新。

### 1. tq (天气查询)
命令行在线查询天气，调用心知天气查询。

```sh
$ python3 ./tq.py  wuhan

2017-10-29 周日; 11°C ~ 21°C; 多云=>多云

$ chmod a+x ./tq.py
$ tq.py beijing

### 2. fy (翻译工具)

```
